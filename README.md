#  Data Science Salary Analysis 2025

> Analyzing 45,000+ Data Science salaries 
> to uncover trends and predict compensation

---

##  Dataset Overview
- Source: DataScience_salaries_2025.csv
- Rows: 45,523
- Years: 2020-2025
- Features: Experience, Job Title, 
  Company Size, Remote Ratio

---

##  Key Insights
1. Applied Scientist pays highest (~$185,000)
2. Experience has 90% impact on salary
3. Remote = On-site salary (almost equal)
4. Medium companies pay more than Large!
5. Qatar offers highest salaries globally
